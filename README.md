# Autohotkey-Script
=> My Autohotkey config to use Win10 as i3wm :) <=
- Download AutoHotkey here: https://www.autohotkey.com/
<br><H3>AutoHotKeys bindsym</H3>
- `Win+b` Open Edge
- `Win+x` Open Explorer
- `Win+#` Open virtual workspace
- `Win+E` and `Win+D` were "delete"
